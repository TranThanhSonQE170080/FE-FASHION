# E-Commerce Platform - Design Upgrade Plan

## Design Guidelines

### Design References (Primary Inspiration)
- **Nike.com**: Bold product imagery, clean layouts, modern minimalism
- **ASOS.com**: Grid-based product displays, smooth interactions
- **Zara.com**: Premium feel, elegant typography, sophisticated color palette
- **Style**: Modern E-commerce + Premium Fashion + Clean Minimalism

### Color Palette
- Primary: #000000 (Pure Black - headers, text)
- Secondary: #FFFFFF (White - backgrounds, cards)
- Accent: #FF6B6B (Coral Red - CTAs, highlights, hover states)
- Text Primary: #1A1A1A (Almost Black)
- Text Secondary: #6B6B6B (Medium Gray)
- Border: #E5E5E5 (Light Gray)
- Background: #F8F8F8 (Off-white)

### Typography
- Heading1: Inter font-weight 700 (48px) - Product titles, hero text
- Heading2: Inter font-weight 600 (36px) - Section headers
- Heading3: Inter font-weight 600 (24px) - Card titles
- Body/Normal: Inter font-weight 400 (16px) - Descriptions
- Body/Small: Inter font-weight 400 (14px) - Secondary info
- Button: Inter font-weight 600 (16px) - CTAs

### Key Component Styles
- **Product Cards**: 
  - White background with subtle shadow on hover
  - Image aspect ratio 3:4 (portrait)
  - Smooth scale transform on hover (1.02x)
  - Rounded corners 12px
  - Padding 16px
  
- **Buttons**: 
  - Primary: Black background, white text, hover: coral accent
  - Secondary: White background, black border, hover: black background
  - Border radius 8px
  - Padding 12px 32px
  - Smooth transitions 300ms

- **Navigation**:
  - Sticky header with backdrop blur
  - Clean horizontal menu
  - Search bar with icon
  - Cart icon with badge

- **Product Grid**:
  - 4 columns desktop (1fr each)
  - 3 columns tablet
  - 2 columns mobile
  - Gap 24px
  - Masonry-style layout for visual interest

### Layout & Spacing
- Hero section: 600px height with full-width image
- Section padding: 80px vertical, 120px horizontal (desktop)
- Card spacing: 24px gaps
- Content max-width: 1400px
- Smooth scroll behavior
